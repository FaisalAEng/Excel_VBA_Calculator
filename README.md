# Excel VBA Calculator
**Automating Repetitive Engineering Calculations in Excel**

![Excel VBA Calculator](https://raw.githubusercontent.com/FaisalAEng/portfolio-website/refs/heads/main/assets/images/projects/excel_vba_calculator.png)

## Project Overview
This project involved developing an Excel-based calculator using VBA to automate repetitive engineering calculations. The goal was to reduce manual computation time, improve accuracy, and provide a user-friendly interface for performing multi-step calculations commonly required in engineering workflows.

The project emphasizes automation, data validation, and user interaction in Excel, providing a practical tool for real-world engineering tasks.

## System Description
The Excel VBA calculator consists of:
- A structured workbook with input, calculation, and output sheets.
- Custom VBA macros to perform calculations automatically.
- Data validation to ensure accurate inputs.
- User forms for easy input and navigation.

Key functionalities include automated multi-step calculations, error checking, and dynamic output generation, reducing manual effort and improving workflow efficiency.

## My Contributions
- Designed the overall workbook structure and layout.
- Developed VBA macros for automating calculations.
- Created user forms for intuitive data input.
- Implemented data validation and error handling to improve reliability.

## Observations & Performance
- Automated previously manual calculations, reducing effort by ~80%.
- Improved accuracy by eliminating manual data entry errors.
- Provided clear and organized output for decision-making.
- User interface allows quick modification of input values and recalculation.

## Outcome
This project provided hands-on experience with Excel automation, VBA programming, and workflow optimization. It demonstrates the value of combining engineering knowledge with programming to streamline repetitive tasks and increase productivity.
